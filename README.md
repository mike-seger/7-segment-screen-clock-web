# 7-segment-screen-clock-web

This is a simple 7-segment clock for the web.

<img src="screens/960x540.png">

Simply open [index.html](https://mike-seger.github.io/7-segment-screen-clock-web/) with your web browser.

<a href="https://mike-seger.github.io/7-segment-screen-clock-web/" target="_blank">Screen Clock</a>


## Run a local webserver in the directory

In order to run the clock with configuration functionality without CORS issues, you need to run a local web
server.

```
python3 ./no-cache-server.py
```
